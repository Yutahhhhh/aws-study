# AWS学習ビジュアライザー

AWSの通信・ネットワーク周りの概念を、インタラクティブなビジュアルで理解する学習プラットフォーム

## 🎯 機能

- **ネットワークシミュレーション**: AWS VPCにおけるパケットの流れを視覚的に理解
- **ステップバイステップ学習**: パケットの各段階での変化を詳細に解説
- **用語集**: AWS/ネットワーク用語を即座に参照できる辞典機能
- **3つのシミュレーションモード**:
  - インバウンド通信(外部→ECS)
  - アウトバウンド通信(ECS→外部、NAT Gateway経由)
  - コスト削減版(ECS→外部、IGW直接接続)

## 🛠️ 技術スタック

- **React 18** + **TypeScript**
- **Vite** (ビルドツール)
- **Tailwind CSS** (スタイリング)
- **React Router** (ルーティング)
- **Lucide React** (アイコン)

## 🚀 セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

開発サーバーは `http://localhost:5173/` で起動します。

## 🎨 設計思想

### 責務の分離

- **データの外部化**: 用語集やシミュレーションデータは独立したファイルとして管理
- **コンポーネントの単一責任**: 各コンポーネントは明確な役割を持つ
- **型安全性**: TypeScriptによる厳密な型チェック

### コンポーネント設計

1. **Presentation Components**: 見た目のみを担当
2. **Container Components**: 状態管理とロジックを担当
3. **Data Files**: ビジネスロジックとデータを分離

## 📚 学習テーマ

### 1. IP変換(NAT) ✕ ルーティング

AWS VPCにおけるパケットフローを視覚化:

- インターネットゲートウェイ(IGW)の役割
- NAT Gatewayによるアウトバウンド通信
- パブリック/プライベートIPの変換
- ルートテーブルによる経路制御
- セキュリティグループの設定
